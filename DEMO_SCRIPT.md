## Recording Tips
1. Use **1920x1080** screen resolution
2. Keep the browser full-screen
3. Use smooth, slow mouse movements between pages
4. Wait 2-3 seconds on each page before narrating
5. The Agent Decision Graph auto-animates — let it run for a few seconds
